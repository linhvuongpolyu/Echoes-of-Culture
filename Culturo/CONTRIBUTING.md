### Development Setup

1. **Fork the repository**
   ```bash
   git fork https://github.com/linhvuongpolyu/Culturo.git
   ```

2. **Clone your fork**
   ```bash
   git clone https://github.com/YOUR_USERNAME/Culturo.git
   cd Culturo/Culturo
   ```

3. **Set up development environment**
   ```bash
   python -m venv culturo_dev
   source culturo_dev/bin/activate  # On Windows: culturo_dev\Scripts\activate
   pip install -r requirements.txt
   ```

4. **Run the application**
   ```bash
   python start_app.py
   ```
